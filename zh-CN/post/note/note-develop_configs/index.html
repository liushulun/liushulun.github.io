4. Android Issues

4.1 AndroidStudio

(1) SDK & NDK

(2) ADB connect device

  • Get the ip address with port (default is 5555) of device
  • Execute: adb connect x.x.x.x:5555

Android simulators can be connected by the same way.

Android devices can be connected through WiFi, by getting ip from WiFi settings on the device. This requires that the computer hasn't connected any other devices or simulators.
